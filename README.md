# Colors

Usando la escala de tailwind css de 50...950

```
colors: {
  inherit: 'inherit',
  current: 'currentColor',
  white: '#ffffff',
  primary: {
    light: 200
    DEFAULT: 600
    container: 50
  },
  surface: {
    DEFAULT: slate 50
    dim: slate 200
  },
  on: {
    primary: {
      DEFAULT: white,
      container: 900 o 950
    },
    surface: {
      DEFAULT: slate 900
    }
  }
}

```
